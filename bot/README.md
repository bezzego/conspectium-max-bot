# Установка зависимостей

## Требования
- Node.js версии 14.0.0 или выше
- npm (устанавливается вместе с Node.js)

## Установка

1. Установите зависимости:
```bash
npm install
```

Это создаст папку `node_modules/` с всеми необходимыми пакетами (аналог виртуального окружения для Node.js).

2. Создайте файл `.env` в корне проекта:
```bash
MAX_BOT_TOKEN=ваш_токен_бота
MAX_WELCOME_MESSAGE=Привет! Я бот Макса. Чем могу помочь?
```

3. Запустите бота:
```bash
npm start
```

или

```bash
node max.js
```

## Обновление зависимостей

Для обновления всех зависимостей до последних версий:
```bash
npm update
```

## Удаление зависимостей

Для удаления всех установленных пакетов:
```bash
rm -rf node_modules/
```
